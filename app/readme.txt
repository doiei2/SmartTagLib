本ドキュメントはスマートタグサンプルプログラム"SmartTagSample"
について説明するものです。


開発環境:

Android SDK
Java開発環境(JDK)
Eclipse + Android Plugin 

※Androidアプリケーション作成のための一般的な環境で使用できます。
※動作確認にはスマートタグ本体が必要です。


実行環境:

Android 2.3.3以降のNFC対応端末(スマートフォン・タブレットPC等)
(おサイフケータイ非対応機種)


プロジェクト環境:

Android SDK API Level 10で作成しています。


動作確認済み端末(2012/1/18現在):

Samsung Nexus S, Galaxy Nexus
Sharp RW-T107
